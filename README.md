## 📖 Overview

AgroVision AI is an intelligent agriculture-based mobile application developed to detect plant leaf diseases using Artificial Intelligence and Computer Vision.

The application allows farmers and agriculture enthusiasts to scan plant leaves using their smartphone camera and instantly receive:

- 🌿 Disease Detection
- 📊 Confidence Score
- 💊 Treatment Suggestions
- 🧪 Recommended Medicines
- 📚 Disease Information
- 🌍 Multi-language Support (English + Sindhi)

The project aims to support smart farming and sustainable agriculture practices through offline AI-powered disease recognition.

# 🧠 AI Model

The AI model is trained on Mango Leaf Disease datasets using Convolutional Neural Networks (CNN) and converted into TensorFlow Lite format for mobile deployment.

### Supported Diseases
- Anthracnose
- Bacterial Canker
- Powdery Mildew
- Die Back
- Gall Midge
- Healthy Leaf Detection
